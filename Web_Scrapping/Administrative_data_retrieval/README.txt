This program is used to get data from consulting web page. It retrieves
data from this website through the action of passing parameters as inputs. 
These parameters must be in a .txt file named "input.txt", delimited by commas and with space between each of them.

The results of data retrieval are inserted into a excel table, ordering the records by a date field.

Furthermore, there is a error.txt file that allows you to see which process failed and react to it.
