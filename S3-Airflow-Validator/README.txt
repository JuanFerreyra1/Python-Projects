This programme was created to validate data over all sources used for that. Every class contains its own validations, and there's one extra class, which name is "Structure", that gives us needed json structure to perform all operations.

Airflow module: Useful for getting information about data in Airflow.
Adjustment module: Useful for getting information about data in platform.
S3 module: Useful for getting information inside AWS S3 buckets.
Structure module: Useful for getting needed structure to generate the final output.
